# library
Library Portal 

The following options are given to the user:

1 Manage account

 1.1 login
 
 1.2 New User
 
 1.3 logout

2 Search catalog

 2.1 search by name
 
 2.2 search by author


3 Reserve book

4 Release Book

User can be either admin or borrower.
Only admin has the right to update the database of books.
User can only issue or search for a book.
